<h1>Портфолио аналитика данных</h1>
<br>
  Данный репозиторий содержит исходный код персонального сайта-портфолио аналитика данных.
Сайт реализован как многостраничный статический веб-проект на HTML, CSS, JavaScript и предназначен для демонстрации проектов, навыков и профессионального профиля.

### Структура сайта

## Главная страница — `index.html`

Главная страница является точкой входа и включает следующие блоки:

## 1. Header
- Логотип
- Навигационное меню
## 2. Основной блок 
- Краткое позиционирование
- Визуальный акцент
## 3. Обо мне
- Профессиональный профиль
- Навыки
- Кнопка для просмотра полного резюме
## 4. Проекты
- Ссылки для перехода на отдельные страницы проектов
## 5. Контакты
- Способы связи
- Ссылки
## 6. Footer
- Логотип
- Навигационное меню (только в desktop версии)

## Страницы проектов — `projects`

Проект включает отдельные страницы для каждого направления:

- `project-1.html`
- `project-2.html`
- `project-3.html`

На текущем этапе данные страницы содержат заглушку  
**«Раздел в разработке»** и будут дополнены реальными проектами, кейсами и визуализациями по мере развития портфолио.

## Дизайн и визуальная концепция

В проекте используется авторская графика:

- Иллюстрация робота в разных позах
- Изображения созданы с помощью инструментов искусственного интеллекта
- Робот выступает как визуальный символ аналитики, технологий и автоматизации

Графика используется исключительно в рамках данного портфолио.

## Технологии

- **HTML5** — структура страниц  
- **CSS3** — стилизация и адаптивная верстка  
- **JavaScript** — базовая интерактивность  
- **GitHub Pages** — хостинг сайта

## Многоязычность

Англоязычная версия портфолио размещена в отдельном репозитории.  
На данный момент она находится в стадии заготовки (болванки) и будет развиваться параллельно с русской версией.

## Статус проекта

Проект находится в активной разработке.  
Структура сайта сформирована, контент и проекты будут постепенно дополняться и обновляться.

## Назначение проекта

Сайт предназначен для:

- демонстрации портфолио аналитика данных
- представления проектов и кейсов
- профессионального позиционирования

## Контакты

Контактная информация и актуальные ссылки размещены на сайте в разделе **«Контакты»**.
